Password Security Analyser

Final Year Project • Cybersecurity Tool

Project Overview

The Password Security Analyser is my final year project focused on improving password security awareness. This comprehensive tool combines multiple security analysis techniques to provide users with detailed feedback about their password strength and security vulnerabilities.

Key Features

• Advanced password strength analysis using the ZXCVBN algorithm
• Malaysian cultural pattern detection for localized security insights
• Have I Been Pwned (HIBP) database integration for breach detection
• OpenAI API integration for intelligent security recommendations
• Real-time feedback and security score calculation
• User-friendly web interface with detailed analysis reports

Technical Implementation

Built using Python Flask framework with a responsive frontend, the application integrates multiple APIs to provide comprehensive password analysis. The system processes passwords locally for privacy while connecting to external databases for breach detection.

Smart Fish Feeder System

IoT Automation • Pet Care

Project Overview

An automated fish feeding system designed to provide consistent and reliable feeding schedules for aquarium fish. Built using Arduino Uno with servo motor control and I2C display technology for precise timing and monitoring.

Key Features

• Arduino Uno-based control system for reliable operation
• Servo motor for precise food dispensing mechanism
• I2C LCD display for schedule monitoring and system status
• Programmable feeding schedules with multiple daily feeds
• Portion control system to prevent overfeeding
• Manual override functionality for immediate feeding
• Low food level detection and alerts

Hardware Design

The system features a custom-designed food hopper with servo-controlled dispensing mechanism. The I2C display provides real-time feedback on feeding schedules, system status, and maintenance alerts. The Arduino Uno ensures reliable operation with minimal power consumption.

Benefits

This automated system ensures consistent fish care even when owners are away, prevents overfeeding, and maintains optimal aquarium health. The programmable interface allows for customization based on different fish species requirements.
